# 📖 読書DNA＋スマートダイジェスト（v3 未着手モジュール）

> 別の機会に実装する。優先度・進め方は着手時に再確認。

## 機能の柱

1. **4軸プロファイリング** — 行動から嗜好を学習
   - 深さ: 理論派 ↔ 実装派
   - 時制: 近未来志向 ↔ 長期トレンド志向
   - 視点: 研究者 ↔ エンジニア ↔ ビジネス
   - 広さ: 専門特化 ↔ 広範収集
   - 開封・サマリー展開・お気に入りごとに重み付けを変えて反映
2. **知的変化の記録** — 「3ヶ月前はLLM推論中心→先月からエージェントへ」を可視化、最近読んでない分野を検知して意識的にカバーを促す
3. **サマリーのトーン適応**
   - 実装派 → 「実際にどう使うか」「コードはどこにある」を前面に
   - 理論派 → 「どんな仮定か」「先行研究との違い」を前面に
   - ビジネス派 → 「市場への影響」「採用コスト」を前面に
4. **スマートダイジェスト5形式**
   | フォーマット | タイミング | 内容 |
   |---|---|---|
   | Morning Flash | 平日朝 | 重要5件、各1行。30秒で読める |
   | Deep Dive | 週末 | 2000字。今週の流れを物語として |
   | Topic Brief | オンデマンド | 特定テーマに絞った完全総括 |
   | Learning Recap | 毎週日曜 | 「今週あなたが学んだこと」の振り返り |
   | Prediction Brief | 月初 | 先月データから見える今月の展開予測 |
5. **会話への誘導** — メールの各記事に「この記事について話す」リンク → AIが3つの問いを用意した状態でチャット起動

## 既存資産の活用

- `userTopicWeights` テーブルと `markAsRead`/`toggleFavorite` の重み加算ロジック（4軸はこの発展形）
- メール基盤（`sendEmail`/`markdownToHtml`、日次/週次/月次レポート）→ 5形式はこの拡張
- チャット（`/api/chat`/ChatPanel/MobileChatModal）→ 深掘りリンクの接続先

## 新規実装

- **スキーマ**: `reading_events`(行動ログ: open/expand/favorite + 各軸への寄与)、`reading_profile`(4軸スコア + 履歴スナップショット)
- **行動トラッキング**: サマリー展開も記録（現状は展開がクライアント内のみで永続化されていない）
- **パイプライン**: プロファイル算出 / 知的変化スナップショット / 各ダイジェスト生成
- **UIタブ**: 4軸レーダーチャート + 関心シフトの時系列
- **チャット深掘りリンク**: 記事コンテキスト付きでチャットを開く

## 依存/前提・コスト・規模

- **依存**: なし（既存テーブルの自然な発展）
- **コスト**: ダイジェスト形式が増える分Gemini呼び出し増。トーン適応はサマリー再生成 or プロンプト分岐で増える
- **規模**: 大（行動ログ基盤＋プロファイル＋5形式メール＋UI）。最も変更箇所が多い。サブ機能ごとに段階分割推奨
