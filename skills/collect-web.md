# Skill: Web収集 (collect-web)

## 目的
現在アクティブな検索キーワードを用いてWeb（Zenn, Qiita等）から最新の技術情報を収集し、データベースに保存する。

## 手順
1. **キーワードの取得**:
   - `sources` テーブルから `status = 'active'` かつ `type = 'keyword'` のレコードを全件取得する。
   - レコードが存在しない場合は、`status = 'candidate'` のものから上位3件を取得して使用する。

2. **Web検索の実行**:
   - 取得した各キーワードについて `WEB_SEARCH` ツールを実行する。
   - 特に `Zenn` や `Qiita` のドメインを優先して検索するように指示する。

3. **データの保存**:
   - 検索結果から「タイトル」「URL」「概要」を抽出する。
   - `collected_data` テーブルに保存する。
   - `url` が既に存在する場合はスキップする。
   - 保存に成功したソースの `last_hit_at` を現在時刻に更新する。

4. **完了報告**:
   - 新規に収集された件数を報告して終了する。
